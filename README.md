# SAE-Projet-datasets-impact-IA
Analyse Statistique de l’Impact Global de l’IA sur Divers Secteurs d’Activité (2020-2025)

> Projet réalisé dans le cadre du BUT Science des Données — 1ère année

---

## Description

Ce projet a pour objectif d'analyser un jeu de données mesurant l’impact de l’IA sur divers secteurs à l'aide du langage **R**.
Il comprend le nettoyage des données, l'exploration statistique et la visualisation des résultats structuré en 4 lots.

## Prérequis

- [R](https://www.r-project.org/) (version 4.0 ou supérieure)
- [RStudio](https://posit.co/download/rstudio-desktop/) (recommandé)
- Packages R nécessaires :

```r
install.packages(c("tidyverse", "ggplot2", "dplyr"))
```

---

## Lancer le projet

1. Clone le repository :
```bash
git clone https://github.com/vilcentpricild-rgb/SAE-Projet-datasets-impact-IA
```

2. Ouvre RStudio et charge le script principal :
```r
source("scripts/Script_R_SAE_Projet.R")
```
